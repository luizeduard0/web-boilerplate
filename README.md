# Reax

A boilerplate for building React and Redux apps with ES6

## How to install?

`npm install -g webpack webpack-dev-server && npm install`

## How to build?

### Production

`webpack -p`

### Development

`BUILD_DEV=1 webpack-dev-server --content-base public/ --inline`

## How to run?

### Production

`PORT=80 node server.js`
